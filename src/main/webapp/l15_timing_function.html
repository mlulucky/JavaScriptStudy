무한 반복문 setTimeout 과 setInterval 을 알아보자!

setInterval(callBackFunc,milliseconds) : => return threadNum(새로생성된 일꾼번호) 을 반환 밀리초마다 콜백함수를 호출하는 무한 반복문(멀티스레드)

setInterval 로 시계 만들기

clearInterval(threadNum) : setInterval 로 생성된 스레드를 삭제하는 함수(일하는 걸 없애는 것)

setTimeout(eventHandler,milliSec) return threadNum : 해당 밀리초 동안 대기하고 있다가 이후에 콜백함수를 실행하는 무한 반복문(멀티스레드). 끝이 있다. (<-> setInterval 은 끝이 없다)
